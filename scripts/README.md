# scripts

Hilfsskripte für Entwicklung, Build und Tests.

Typische Skripte im Projekt-Root `scripts/` und `backend/scripts/`:
- `generate-client.sh` — erzeugt den Frontend-Client aus der OpenAPI-Spezifikation.
- `init-dev.sh`, `test.sh`, `build.sh` — Entwicklungs- und Test-Helfer.

Achte auf ausführbare Rechte (`chmod +x`) bei Skripten, die direkt ausgeführt werden.
