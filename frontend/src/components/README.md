# frontend/src/components

Enthält wiederverwendbare UI-Komponenten (Chakra UI-basiert).

Konventionen:

- Komponenten in eigenen Unterordnern mit `index.tsx` und ggf. `styles.tsx`/`types.ts`.
- Kleine, fokussierte Komponenten sind vorzuziehen.

Tests & Storybook:

- Wenn vorhanden, pflege zugehörige Tests unter `frontend/tests/` oder eine Storybook-Config.
